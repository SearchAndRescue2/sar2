Hardware requirements:

Pentium 166 or faster (minimally playable on a P166 w/o 3D accel and only object texturing enabled)
16 MB minimum, 32 MB or more recommended for more complex scenery
Video card with 3D acceleration (optional)
Joystick (optional, 3D stick recommended)

Software requirements:

Implementation of OpenGL (such as Mesa3D)
DRI Direct rendering (if using hardware acceleration)
libjsw (optional for joystick support)
YIFF (optional for sound support)
DirectX 8.0 or newer (for Win32 version)
