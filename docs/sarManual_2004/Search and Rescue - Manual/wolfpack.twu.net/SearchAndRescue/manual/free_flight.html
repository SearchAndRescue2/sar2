Starting Free Flight

  1. Make sure you are at the Main Menu and click on Free Flight. Then you will see a list of Theaters (scenery) and a list of starting Locations.
  2. Select a Theater (scenery), the starting Locations for that Theater will be displayed on the list below it.
  3. Select a starting Location and then click on Begin. Then you will see a list of Aircrafts.
  4. Select an Aircraft and then you can either begin the simulation by clicking on Begin or select the weather by clicking on Weather...
  5. If you clicked on Weather... then you can select the startup Weather Condition and then click on Begin.

At this point you should see the Loading Simulation screen and afterwards you will enter simulation.

To learn more about flying the aircraft see Flying.

For a complete list of keyboard keys see Keyboard Keys List

Aircraft Information

While in the Aircrafts Menu you can get information about a specific aircraft by selecting an aircraft and then click on Details.... You should then see the Aircraft Details Menu, now you can use the pointer or keyboard to change the aircraft's direction and other viewing aspects. The following are a list of keys and their function in the Aircraft Details Menu:

  • CURSORS - Change aircraft heading pitch.
  • CTRL + CURSORS - Change aircraft bank.
  • SPACE - Rotate aircraft about its heading azis.
  • BACKSPACE - Return to default rotation.
  • PLUS - Zoom in.
  • MINUS - Zoom out.
  • M - Toggle messages.
  • B - Cycle background color.
  • L - Toggle lighting.
